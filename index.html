A visualization of array methods.
Original array: const cars = ["BMW", "Volvo", "Alfa Romeo", "Fiat"];

You can choose what element from an array to show. (cars[0], cars[1], cars[2]... indexing)

Using cars[i], you can change that particular element very easily.
cars[i] = "Honda"

Believe it or not, you can actually add an element using .length
cars[cars.length] = "Audi";

You can merge multiple arrays using the concat() method.
Let's say we have two arrays of cars:
Through the concat() method, we will say:
const carChildren = cars1.concat(cars2);

And now; an example with multiple arrays:

const carChildren = cars1.concat(cars2, cars3, cars4, cars5);

Using the splice() method, you can more accurately decide where to add new items to an array.
It looks like this:
cars.splice(2, 0, "Nissan", "Mazda");
The first parameter defines where it should be added (index 2), while the second parameter defines how many elements should be removed.
Click the button under to see what it does to our array.

In this theme, you can remove elements with the splice() method.
cars.splice(0, 1);
This will remove the first element (but you can specify!). Test it under here:

Using the slice() method, you can "slice" out one element that gets put in a new array.
It doesn't remove any elements from the source array, and it looks like this:
const slicedOutPart = cars.slice(1);
This will "slice" out the first element, and create a new array from the leftovers, as you can see here:

You can also use multiple arguments here too, something like this:
cars.slice(1, 3);
This one will cut out a part from index 1, up to (BUT NOT INCLUDING) index 3!
